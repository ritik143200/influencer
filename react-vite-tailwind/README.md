# ViralMantrix Frontend

This frontend powers the public ViralMantrix website, creator onboarding flow, brand inquiry flow, and dashboard entry points.

## Scope

The frontend is being refocused from a generic artist platform into a creator-campaign experience for:

- influencers
- celebrities
- city pages
- meme pages
- creator communities
- brands

## Key Frontend Areas

### Public Website

- homepage hero and featured profiles
- services and how-it-works storytelling
- category discovery
- sign-up and inquiry entry points

### Creator Flow

- registration and onboarding
- category and micro-category selection
- social media connection
- dashboard access

### Brand Flow

- campaign inquiry form
- multi-category hiring selection
- micro-category selection
- follow-up dashboard access

## Stack

- React
- Vite
- Tailwind CSS
- Lucide React

## Scripts

```bash
npm run dev
npm run build
npm run preview
npm run lint
```

## Default Local URL

```text
http://localhost:5173
```

## Current UX Direction

The current frontend work is aimed at matching the provided prototype more literally:

- soft lavender and purple visual system
- rounded white cards
- light product-style onboarding forms
- campaign-first homepage messaging
- featured profile presentation

## Important Note

Changes are previewed locally before anything is pushed or prepared for deployment.
